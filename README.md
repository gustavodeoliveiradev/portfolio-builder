O **Portfolio Builder** é uma aplicação web que permite criar um portfólio profissional de forma rápida e intuitiva. Preencha o formulário com seus dados, adicione seus projetos, escolha um tema e gere seu portfólio personalizado.

## 🚀 Funcionalidades

- ✅ **Formulário Intuitivo**: Insira suas informações pessoais, bio, habilidades e projetos
- ✅ **Múltiplos Projetos**: Adicione quantos projetos quiser ao seu portfólio
- ✅ **3 Temas Disponíveis**:
  - 🎨 Minimalista
  - 🔥 Moderno
  - ✨ Elegante
- ✅ **Preview em Tempo Real**: Visualize como ficará seu portfólio antes de finalizar
- ✅ **Design Responsivo**: Funciona perfeitamente em dispositivos móveis e desktop
- ✅ **Ícones Font Awesome**: Interface moderna e atraente

## 📖 Como Usar

1. **Abra o projeto**: Basta abrir o arquivo `index.html` em seu navegador
2. **Preencha o formulário**:
   - Nome completo
   - Bio/sobre você
   - Email e localização
   - Habilidades (separadas por vírgulas)
   - Projetos (título, link e descrição)
3. **Escolha um tema**: Selecione entre Minimalista, Moderno ou Elegante
4. **Gere o preview**: Clique em "Gerar Preview" para visualizar
5. **Adicione mais projetos**: Use o botão "Adicionar Projeto" para incluir quantos precisar
